/////////////////////////////////////////////////////////////////

オブジェクトプーリングに移行するにあたり、使い方を残します。

////////////////////////////////////////////////////////////////

① 各オブジェクトは、ゲーム開始時に必要数生成され、生成された瞬間に機能を停止(動かなくなるけどゲーム中に存在はしている)。

② 機能停止したオブジェクトは別スクリプトで再起動して使います。

③ Object_Instanクラスにある関数を使うと再起動が簡単だと思います。再起動時に、名前とポジションと向きを引数に入れてね。

再起動に使う Object_Reboot関数 の使い方
//----------------------------------------------------------------------------------
// using StorageReference;
// をいつもの場所に定義してください。
// 
// 使いたいタイミングで Unity の Instantiate(・・・) のように
// Object_Instantiation.Object_Reboot( 呼び出したいObjectの名前, ポジション, 向き)
// 呼び出してくださいね！！
//----------------------------------------------------------------------------------
